# Student Record Management System (SRMS)

## Project Overview
This system is a Java-based application developed for Qassim University to manage student records efficiently. It integrates advanced programming concepts including Object-Oriented Programming (OOP), File Management, and Multithreading.

## How the Project Works
The SRMS provides a command-line interface for university staff to:
1.  Manage Student Data: Create and update student profiles with specific attributes.
2.  File Persistence: All records are automatically stored in text files, ensuring data is not lost after closing the program.
3.  Advanced Reporting: Utilize multithreading to generate background reports without interrupting the main application flow.
4.  Error Prevention: Comprehensive Exception Handling ensures the system handles invalid inputs or missing files gracefully.

##  Instructions for Compiling and Running
To run the system on your local machine, follow these steps:

1.  Compile all Java files:
       javac *.java
    2.  Run the Main application:
       java Main
